# AutoCompleteScript

Excel シート内で最新列の情報を元に**自動補完＆古い自動生成テキストのクリーニング**を行うための自動処理ツール


## 前提条件

* Excel Online（Microsoft 365）の **Office Scripts** が利用可能であること
* データは「左から右へ時系列で列が増えていく」構造になっていること
* 1行目には「見出しや日付」が入っており、右に行くほど新しい列であること

## 設定方法（使い方）

### ① Excel Online を開く

ブラウザで対象の Excel ファイルを開く

### ② メニューから

**「自動化」 → 「新しいスクリプト」**
を押す

### ③ エディタ画面にコードを貼り付ける

上記スクリプトをそのままコピペ

### ④ 保存して名前を付ける

例：
**AutoFill  Script**

### ⑤ 実行ボタン を押す

1クリックで

* 自動補完
* グレーで表示
* 古いグレーの自動生成テキストの削除

## 注意点

* フォント色が `#888888` のセルのみ削除します
  → それ以外は消しません（黒文字などは安全）
* グレー文字の削除が機能しない場合は、Excel が色コードを変換するケースがあります
* 色判定が不安定な場合は "タグ方式" へ変更することもできます
  （必要なら改良版を作成します）
